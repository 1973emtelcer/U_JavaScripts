Cálculo de la factura

Utilizando las clases de Java Script, se va a hacer un equeño programa que calcule el precio total de una factura. Para ello:

  • Crea una clase de factura que recibe el nombre del cliente, un array de elementos y una pseudoclase información con las propiedades: baseImponibe, iva, total y forma de pago.
  • La clase Cliente recibirá el nombre, dirección, teléfono y nif.
  • La clase Elemento recibirá la descripción, cantidad y precio.

Una vez que tengamos las clases haremos las funciones que van a hacer los cálculos y mostrarlos en la pantalla:

  • Cálculo del total: recorreremos los elementos del array de elementos calculando la base imponible de la factura y el total. Esa información la guardaremos en el total de la pseudoclase información.
  • Base imponible = cantidad * precio.
  • Total = base imponible * (1+ (100/16)).
  • Mostrar en pantalla. Ejecutaremos la función anterior y mostraremos el resultado por pantalla.
